# Инструкция по деплою проекта «Зелёный Контур»

Проект построен на стеке **Next.js (App Router)** и требует окружения Node.js версии 18.0.0 или выше.

## 1. Установка и запуск (локально или на VPS)
1. Распаковать архив.
2. Выполнить установку зависимостей: `npm install`
3. Собрать проект: `npm run build`
4. Запустить: `npm run start`

## 2. Настройка переменных окружения (Environment Variables)
Для работы форм обратной связи необходимо создать файл `.env.local` (или прописать переменные в панели хостинга):

| Переменная | Описание | Пример |
|------------|----------|--------|
| `SMTP_HOST` | Адрес почтового сервера | `smtp.mail.ru` |
| `SMTP_PORT` | Порт (обычно 465 или 587) | `587` |
| `SMTP_SECURE` | Использование SSL | `false` для 587 порта |
| `SMTP_USER` | Логин от почты отправителя | `your-email@mail.ru` |
| `SMTP_PASS` | Пароль приложения (не от почты!) | `ваш_пароль_приложения` |
| `MAIL_FROM` | От кого придет письмо | `your-email@mail.ru` |
| `LEADS_TO` | Куда отправлять заявки | `client-email@yandex.ru` |

## 3. Настройка в Timeweb Cloud (App Platform)
Если проект разворачивается через **App Platform**:
1. Тип приложения: **Node.js**.
2. Команда сборки: `npm install && npm run build`
3. Команда запуска: `npm run start`
4. Путь до директории: `/`
5. В разделе **«Переменные»** обязательно добавить все ключи из таблицы выше, иначе формы не будут работать.
